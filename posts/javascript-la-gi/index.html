JavaScript là gì? #

JavaScript (gọi tắt là JS) là ngôn ngữ lập trình chủ yếu dùng để tạo các trang web tương tác và ứng dụng phía client. Bắt đầu từ trình duyệt, JavaScript hiện đã phát triển mạnh mẽ và chạy ở nhiều môi trường khác nhau (ví dụ: Node.js cho server, Electron cho desktop, React Native cho mobile). JavaScript vừa dễ tiếp cận cho người mới, vừa có chiều sâu để xây dựng hệ thống phức tạp.

Tổng quan #

  • Dễ học cho người mới: cú pháp tương đối trực quan, nhiều tài liệu.
  • Hệ sinh thái rộng: framework và thư viện phong phú (React, Vue, Angular, Express…).
  • Đa nền tảng: web, server, mobile, desktop, tooling.
  • Cộng đồng lớn: nhiều package, plugin, tutorial, bài viết, video.

Lịch sử ngắn gọn #

  • 1995: Brendan Eich phát triển ngôn ngữ tại Netscape.
  • ECMAScript là tiêu chuẩn mô tả JavaScript; ES6 (ES2015) là bước ngoặt lớn với nhiều tính năng hiện đại.
  • Tên “JavaScript” mang tính marketing — không phải là Java.

JavaScript chạy ở đâu? #

  1. Trình duyệt (Browser)
    • Thao tác DOM, xử lý sự kiện, giao tiếp mạng (fetch, XMLHttpRequest).
  2. Server (Node.js)
    • Xây dựng API, xử lý I/O, server-side rendering, tooling (build tools).
  3. Mobile
    • Frameworks: React Native, Ionic, NativeScript.
  4. Desktop
    • Electron, Tauri (JS + native).
  5. Các môi trường khác
    • Embedded (ví dụ: IoT), scripting cho công cụ/IDE.
